SCiO Mobile SDK iOS V1.96, Beta Release September 2016

The iOS SDK contains the following projects, sub-modules and libraries:
1. iOS SDK library.
2. Example app using the SCiO Mobile SDK.

Description:
1. This SDK contains the objects required to communicate with both the SCiO Sensor (device) and the SCiO Cloud, and to login and authenticate the end user on the SCiO Cloud.
2. The example app uses the Mobile SDK to show basic functionality and demonstrate how to scan for SCiO devices, perform login, authentication, and analyze SCiO models.

To make full use of the Mobile SDK, knowledge of Objective C and iOS programming paradigms are assumed.
Familiarity with the Consumer Physics SCiO Lab platform is also assumed.

Complete documentation of the Mobile SDK is available at https://dev.consumerphysics.com/category/mobile-api/ios_api/mobile-api-overview-ios/ and the most recent version of the project for iOS to download is available at https://dev.consumerphysics.com/download/.
